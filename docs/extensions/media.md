Embedded media
==============

Use the following markup to embed audio files, video files and YouTube videos: 

Markup
------

```markdown

![](/images/video.mp4)

![](https://www.youtube.com/watch?v=iIZFvNlZHf0)

```

Results
-------

:::well

![](/images/video.mp4)

![](https://www.youtube.com/watch?v=iIZFvNlZHf0)

:::
