Custom containers
=================

Markup
------

```markdown

:::spoiler
This is a spoiler
:::

```

Results
-------

The markup above will be rendered as

```html
<div class="spoiler">
    <p>This is a spoiler</p>
</div>
```

:::well

:::spoiler
This is a spoiler
:::

:::
