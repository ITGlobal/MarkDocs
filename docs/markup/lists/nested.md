Nested lists
============

This page displays usage of **nested lists**

Markup
------

```markdown
1. Item 1
   * Item 1.1
   * Item 1.2
   * Item 1.3
1. Item 2
   1. Item 2.1
   1. Item 2.2
   1. Item 2.3
1. Item 3
   1. Item 3.1
      * Item 3.1.1
      * Item 3.1.2
   1. Item 3.2
```

Results
-------

:::well

1. Item 1
   * Item 1.1
   * Item 1.2
   * Item 1.3
1. Item 2
   1. Item 2.1
   1. Item 2.2
   1. Item 2.3
1. Item 3
   1. Item 3.1
      * Item 3.1.1
      * Item 3.1.2
   1. Item 3.2

:::
