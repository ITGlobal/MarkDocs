Inline HTML
===========

This page describes usage of inline HTML.

Markup
------

```html

<div class="row">
    <div class="col-md-6 bg-success">
        <h1>My Text</h1>
    </div>
    <div class="col-md-3 bg-info">
        <p>
            Paragraph 1
        </p>
        <p>
            Paragraph 2
        </p>
    </div>
    <div class="col-md-3 bg-warning">
        <button class="btn btn-default">
            <i class="fa fa-ok"></i> Button
        </button>
    </div>
</div>

```
