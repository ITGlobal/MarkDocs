Quotation blocks
================

This page describes quote block markup.

Markup
------

```markdown
> # Quote header
> Quote line 1
>
> Quote line 2
>
> ***
>
> Quote list:
> * Item 1
> * Item 2
> * Item 3
```

Results
-------

:::well

> # Quote header
> Quote line 1
>
> Quote line 2
>
> ***
>
> Quote list:
> * Item 1
> * Item 2
> * Item 3

:::
