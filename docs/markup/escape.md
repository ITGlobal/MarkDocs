Escaping
========

This page describes usage of escape character.

Markup
------

    \*not emphasized*
    \<br/> not a tag
    \[not a link](/foo)
    \`not code`
    1\. not a list
    \* not a list
    \# not a heading
    \[foo]: /url "not a reference"

    BUT!: \\*emphasis*


Results
-------

:::well
