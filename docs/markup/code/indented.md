Indented code block
===================

**Note:** Indented code blocks don't support syntax highlighting.

Markup
------

```markdown
Add 4 spaces in the beginning of each line to create an indented code block

    This section will be rendered as code block
    This section will be rendered as code block
    This section will be rendered as code block
    This section will be rendered as code block
```
